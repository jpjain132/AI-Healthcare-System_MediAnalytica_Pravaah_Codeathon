Help & Instructions

📋 How to Use

  1. Enter Patient Information: Fill in age, gender, conditions, and symptoms
  2. Upload Documents: Drag & drop or select medical reports (PDF/Images/Text)
  3. Analyze: Click "Analyze Documents with AI" to process
  4. Review Results: View AI-generated rare disease possibilities with confidence scores

🔑 Getting an API Key

For real AI analysis, you need an OpenRouter API key:

  1. Visit OpenRouter Keys
  2. Sign up for a free account
  3. Generate an API key (starts with "sk-or-v1-...")
  4. Copy the key and paste it in Settings tab
  5. Save settings and restart the application

⚠️ Troubleshooting

  • Python not found: Install Python 3.8+ from python.org
  • Tesseract not found: Install Tesseract OCR for Windows
  • API errors: Check your internet connection and API key validity
  • File processing fails: Ensure files are not corrupted and are supported formats

📞 Support

For issues or questions:

  • Check the console output for detailed error messages
  • Ensure all dependencies are installed
  • Run in demo mode first to verify basic functionality
